getErrors

@NonNull open fun getErrors(): MutableList<Error!>

Information extracted from the Throwable that caused the event can be found in this field. The list contains at least one Error that represents the thrown object with subsequent elements in the list populated from Throwable#getCause(). A reference to the actual Throwable object that caused the event is available through Event#getOriginalError() ()}.
